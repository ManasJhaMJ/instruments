How to play Guitar online?

Use your computer keyboard or click the drum kit to play the virtual drums. You can use the following shortcuts:

    Strings : Use keyboard shortcuts or simply click to play!

  • 1st string : use e
  • 2nd string : use b
  • 3rd string : use g
  • 4th string : use d
  • 5th string : use a
  • 6th string : use s

    • Chords : Use keyboard shortcuts or simply click to play!

  • C chord : use 1
  • D chord : use 2
  • Dm chord : use 3
  • E chord : use 4
